# Skill: 7S Alignment Framework — BioPharm & Media Integration

## What this does
Applies a professional-services-adapted McKinsey 7S framework to the
BioPharm & Media Integration tracker specifically — this is a post-merger
integration (Indegene acquiring BioPharm Communications), and 7S is the
right lens for THAT shape of engagement in a way SWOT or OKR-alignment
aren't: those ask "is this theme winning," 7S asks "is the acquired firm
actually fusing into the parent firm's operating model without the seams
showing to clients." Do not apply this skill to trackers that are not
post-acquisition/integration engagements (e.g. Pfizer PAVE is an ongoing
performance partnership, not an integration — SWOT/OKR-alignment fit it
better).

## The seven elements, adapted to this integration

| Element | What "aligned" looks like for BioPharm & Media | Tracking metric |
|---|---|---|
| **Strategy** | BioPharm's media/comms capability is being cross-sold into existing Indegene accounts (e.g. bundled into large-deal archetypes alongside things like the Pfizer PAVE playbook), not run as a walled-off unit. | Cross-sell pipeline: $ value of joint proposals pitching BioPharm capability into existing Indegene client relationships. |
| **Structure** | BioPharm's practice leads/partners are integrated into Indegene's existing governance (SteerCo, JSC), not running a shadow reporting line. | Span of control: ratio of BioPharm partners/directors to billable BioPharm consultants, benchmarked against Indegene's existing ratio. |
| **Systems** | BioPharm's time tracking, billing, and CRM are consolidated onto Indegene's stack (the TSA — Transition Services Agreement — governs this handoff), not running parallel systems indefinitely. | Time-to-bill: average days from delivered work to invoice, post-TSA-exit vs. during TSA. |
| **Shared Values** | BioPharm's media/comms delivery culture and Indegene's data/tech-integration culture are actively being blended (joint case studies, shared wins), not coexisting as two unmentioned cultures. | Culture pulse: quarterly eNPS split by legacy-BioPharm vs. legacy-Indegene staff — a widening gap is the signal to watch. |
| **Style** | Project delivery methodology is converging (e.g. if BioPharm ran time-and-materials and Indegene runs fixed-fee agile, is there an explicit plan to reconcile, not two methodologies quietly running side by side). | Write-off rate: % of logged hours that can't be billed due to delivery-methodology friction on blended engagements. |
| **Staff** | Named BioPharm SMEs and practice leads who were part of the deal's value are still there and still billable, not attriting quietly post-close. | Regrettable attrition: % of named/flagged BioPharm revenue-generating staff who've left since close. |
| **Skills** | Indegene's existing consultants are being upskilled on BioPharm's media/comms capability (and vice versa), not treating it as a bolt-on nobody outside BioPharm's original team can actually deliver. | Billable utilization: % of available hours on paid client work, tracked separately for cross-trained vs. single-capability staff. |

## Instructions
- Populate all seven elements using only what's actually recorded for this tracker — strategy goals, tactics/execution insights, stakeholders, financials, risks, and raw ingestion events. If an element has no real evidence either way, say so explicitly ("no signal yet on Systems consolidation") rather than inventing filler.
- For each element, state whether it looks ALIGNED, AT RISK, or UNCLEAR, with the specific fact or gap behind that call — never an unsupported generality like "culture is a work in progress."
- Cost/synergy figures already tracked on this tracker (budget/spend/forecast, synergy line items) map most directly to Systems and Strategy — use them when relevant instead of restating them generically.
- Close with a one-sentence "so what": which ONE of the seven elements is the highest-leverage integration risk right now, given the balance of all seven.
- This is a structured positioning lens, not a replacement for the tracker's own Strategy-vs-Outcome cards — when used during synthesis, let it surface gaps or risks the standard synthesis pass might not have framed in integration-specific terms (e.g. a Systems gap that reads as an ordinary "delayed migration" tactic note is actually a 7S Systems misalignment worth flagging distinctly).
