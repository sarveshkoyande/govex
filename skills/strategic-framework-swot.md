# Skill: SWOT Strategic Evaluation

## What this does
Applies a classic Strengths/Weaknesses/Opportunities/Threats framework to a
tracker, grounded strictly in its recorded strategy goals, OKRs, execution
insights, risks, and financials — a structured lens for "how is this theme
actually positioned," not a generic insight.

## Instructions
- Populate all four quadrants using only the supplied tracker data. If a
  quadrant has no clear support in the data, say so explicitly rather than
  inventing filler — an honest "no strong evidence of X" is more useful than
  a padded list.
- Strengths/Weaknesses are internal (what the tracker's own execution shows:
  cost synergies realized, retention, delivery discipline, vs. gaps in
  execution, unresolved risks, missed targets).
- Opportunities/Threats are external or forward-looking (market/competitive
  signals mentioned in raw events, upside pipeline, dependency risk, a
  looming deadline).
- Every bullet must cite the specific fact behind it (a number, a named
  risk, an insight) — no unsupported generalities like "strong momentum."
- Close with a one-sentence "so what": the single highest-leverage move this
  SWOT implies, given the balance of the four quadrants.
